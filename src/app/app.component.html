Rendering XML strings to Angular

Goal: Rendering any XML strings containing known components to Angular!

Predefined Components

There are two components available for this example app.

bla 

<app-bla>hello</app-bla>

will yield:

hello

blub

<app-blub title="some title">world</app-blub>

will yield

world

Mixing the components

The components can be mixed: 

<app-blub title="some title">bla <app-bla>hello</app-bla> bla</app-blub>
should yield bla hello bla

Goal: Dynamically compile a string to "Angular"

Let's assume now the Angular application doesn't know at startup in which combinations the components will occur. I need a way to compile such dynamic compositions of components.

So I need a function similar to compileToAngular(angularString).

Objective

Implement a function like that, such that I could input any string in the textarea below, click compile, and it would render the Angular correctly.
